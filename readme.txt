Adds a widget for displaying the latest content from an idno feed.

== Description ==

This plugin provides a widget that can display the latest content from your idno
site, friendfeed style.

== Installation ==

1. Upload /wp-content/plugins/ directory
2. Unzip
3. Activate through the plugins menu
